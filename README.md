-------
Prerequisites
-------

```Node v7.1.0```

```Gulp CLI and Local v3.9.1```
                                                                             
-------
Install
-------

```$ npm install```


You have to run this command only once only if gulp is not installed.

```$ npm install -g gulp```

----------
How to run
----------

development:

```$ gulp```

To clean 

```$ gulp clean```

-------

[ DEVELOPMENT ]
This command will run a node server pointing to 8080 port

```$ node server.js```
